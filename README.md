# 🍔 Food Delivery Web Application

> 🚧 **Project Status:** Under Development

A modern full-stack Food Delivery Web Application built using the MERN Stack. This project aims to provide customers with a seamless food ordering experience while enabling administrators to efficiently manage restaurants, menus, orders, and users.

---

## 📖 Overview

This application allows users to browse restaurants, explore menus, place orders, and securely complete payments. An integrated admin panel enables efficient management of restaurants, food items, users, and customer orders.

The project is currently under active development, with new features being added regularly.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/KolliChaithraJyotshna/food-del.git
```

### Navigate

```bash
cd food-del
```

### Install Dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

Admin

```bash
cd admin
npm install
```

---

## ▶️ Run Project

Frontend

```bash
npm run dev
```

Backend

```bash
npm run server
```

Admin

```bash
npm run dev
```
